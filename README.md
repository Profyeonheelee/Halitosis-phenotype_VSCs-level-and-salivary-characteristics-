Analysis-ready dataset for halitosis/VSC study	

Source sheet	Matched_Only
Primary analysis cohort	Rows with VSC_ppb > 0 after data cleaning
Main clinical grouping	Clinical_Halitosis based on the clinical diagnosis field containing Halitosis or hailtosis
Objective VSC grouping	VSC_halitosis based on sex-specific total VSC cutoffs
VSC cutoffs	Female: VSC ≥ 65.79 ppb; Male: VSC ≥ 79.94 ppb
De-identification	Patient name and original chart/medical record numbers were not included. Patient_Key_hash is a hashed linkage key.
Suggested use	Use Analysis_Data for Table 1, regression models, VSC-halitosis analyses, CH3SH/H2S analyses, and salivary-function analyses.
